Vue Graphics Grammar is a library for componentized data visualization implemented in the Vue framework. The library adapts the original grammar first proposed by Hadley Wickam in his paper A Layered Grammar of Graphics and allows for the creation of flexible, customized graphs through different combinations of components.

The github repo is not yet public, but preliminary documentation with example implementations is available here.

The grammar of graphics seeks to 'move beyond named graphics' and gain deeper insight into the steps that go into the creation of these graphics. These steps generally start with some raw data, which are then mapped onto visual aesthetics (x, y positions, color, size etc.). These visual aesthetics are used to create geometries (points, rectangles, lines), which when combined with faceting, statistical transformations, coordinates and themes can be used to create a wide range of graphs without being restricted to a few common data visualizations.

Order of layers in the grammar of graphics, starting with raw data and working upwards to a final visualization.

This library adapts the grammar of graphics by creating Vue components that can be combined to create different kinds of graphs. Broadly speaking, components come in the following categories: Aesthetics, Layers (or Geometries), Transformations, Axes and general plot elements such as legends and titles.

As an example, a scatterplot with regression line is a combination of the following components: point layer, regression transformation, single line layer and the axes.

So far, the Vue Graphics Grammar library has been used in a number of internal lab projects. One of these projects involved replicating a 1985 paper on graphical perception by Cleveland and McGill. All visualizations used in the experiment were created using the library.

Graphs used in the experiment to test human ability to distinguish and compare visual differences (angle versus length versus area).


The library is also being used to create an explorable explanation for another project on global interest in fashion. Readers can toggle data to view differences in regional interest in brands and influencers.
